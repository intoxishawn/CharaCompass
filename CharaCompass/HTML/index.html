O que é o CharaCompass?

O CharaCompass é uma plataforma de interface simples para aqueles usuários interessados em salvar perfis para os seus personagens e mundos. Escreva sobre quem são e onde as aventuras dos seus heróis originais acontecem e ponha a sua criatividade para fora. Esse é o seu mundo!
